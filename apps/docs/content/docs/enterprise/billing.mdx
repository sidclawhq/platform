# Licensing

## SDK — Apache 2.0 (Open Source)

The TypeScript SDK (`@sidclaw/sdk`) and Python SDK (`sidclaw`) are both licensed under **Apache 2.0**. They are free to use in any project, commercial or open-source. No license fees, no usage limits.

- **npm:** `npm install @sidclaw/sdk`
- **PyPI:** `pip install sidclaw`

## Platform — FSL-1.1

The SidClaw platform (API, dashboard, background jobs) is licensed under the [Functional Source License 1.1](https://fsl.software) (FSL-1.1). This license converts to **Apache 2.0** on March 22, 2028.

## Self-Hosting

SidClaw is fully self-hostable. Self-hosting includes:

- Full platform (API, dashboard, background jobs)
- Docker Compose for quick deployment
- Railway one-click deploy template

See the [Self-Hosting Guide](/docs/enterprise/self-hosting) for deployment instructions.
