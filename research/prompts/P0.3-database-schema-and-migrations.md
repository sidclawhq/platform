### 2. Create Prisma Schema (`apps/api/prisma/schema.prisma`)

Map every shared type to a Prisma model. Use the exact field names from `packages/shared/src/types/`. Key rules:

- All `id` fields: `String @id @default(uuid())`
- All enum fields: store as `String` (not Prisma enum) — this keeps the schema flexible and avoids migration pain when enums change. Validate at the application layer via Zod.
- All JSON fields (`authorized_integrations`, `metadata`, `conditions`, `settings`, `onboarding_state`, `scopes`): use `Json` type
- All date fields: `DateTime` with `@default(now())` where appropriate
- `tenant_id` on every model except `Tenant` itself — with `@@index([tenant_id])`

**Models to create:**

#### Tenant

```prisma
model Tenant {
  id                String   @id @default(uuid())
  name              String
  slug              String   @unique
  plan              String   @default("free")

  created_at        DateTime @default(now())
  updated_at        DateTime @updatedAt

  policy_rules      PolicyRule[]
  api_keys          ApiKey[]
}
```
